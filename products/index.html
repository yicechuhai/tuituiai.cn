RK3588 产品安装指南

1. 工业AI视觉工具包

一键安装

git clone https://github.com/yicechuhai/rk3588-industrial-toolkit.git
cd rk3588-industrial-toolkit
pip3 install rknn-toolkit-lite2 numpy opencv-python fastapi uvicorn
sudo cp deploy/systemd/*.service /etc/systemd/system/
sudo systemctl enable rk3588-pipeline rk3588-dashboard
sudo systemctl start rk3588-pipeline rk3588-dashboard
curl http://localhost:8080/api/stats

2. 工业协议实时中间件

源码安装

git clone https://github.com/yicechuhai/rk3588-industrial-rt-middleware.git
cd rk3588-industrial-rt-middleware
sudo bash src/rt-core/cpu_isolation.sh isolate 4-7
sudo python3 src/monitor/jitter_monitor.py -d 60

3. 国产OS适配套件

兼容性检测 + 认证

git clone https://github.com/yicechuhai/rk3588-domestic-os-kit.git
cd rk3588-domestic-os-kit
sudo python3 src/driver-layer/kylinos_compat.py --setup --verify
sudo python3 src/cert/neocertify_runner.py
sudo bash src/peripheral/one_click_install.sh

系统要求

硬件

RK3588 (LubanCat-5 / Orange Pi 5 / NanoPC-T6)
8GB+ RAM, NPU /dev/dri/renderD129

软件

Ubuntu 22.04 / Debian 12 / KylinOS V10 / UOS V20
Python 3.10+, librknnrt.so, OpenCV 4.5+
